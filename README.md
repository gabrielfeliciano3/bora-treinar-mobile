# BoraTreinar Recife - Mobile

Aplicativo mobile desenvolvido com React Native e Expo, focado no incentivo à prática de exercícios nos polos esportivos públicos do Recife.

## Funcionalidades
- **Listagem de Academias**: Consumo da API pública de Dados Abertos da Prefeitura do Recife.
- **Geolocalização**: Cálculo de distância em tempo real entre o usuário e os polos esportivos.
- **Navegação**: Menu inferior (Bottom Tabs) para fácil acesso às áreas de Academias, Histórico e Perfil.
- **Interatividade**: Sistema de Check-in com persistência de dados no backend customizado e integração com Google Maps.

## Tecnologias
- React Native e Expo
- React Navigation (Bottom Tabs)
- Axios (para requisições HTTP)
- Expo Location (para geolocalização)
